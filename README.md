Apache2 / Mariadb / PHP 8.1 + modules / phpMyAdmin / nodeJS / SSL Let's Encrypt / glances

	wget https://is.gd/KlTGhE -O LAMP.sh && chmod +x LAMP.sh
 
 

ثم قم بتشغيله عن طريق الامر التالي 

	./LAMP.sh
